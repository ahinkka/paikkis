Paikkis
=======

MML:n nimistöaineisto ja automaattitäydennys sqlitellä.

Käyttöohjeet
------------

::
    
    # Luo tietokanta (kestää mulla vartin)
    ./paikkis build
    # Tee haku (full-text -indeksi)
    ./paikkis search hakusana(t)
